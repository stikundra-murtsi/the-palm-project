# The Palm Project

The Palm Project - это мой учебный сайт, смысл сайта в том что это как бы компания которая производит и продает пальмы.

Да, на этом сайте используется ИИ, но только для генерации самих пальм, все остальное я писал вручную.
ИИ использованные на сайте:
- ChatGPT - Использовался для создания описания пальм на основе моего юмора с "The sigma project".
- Gemini - Использовался для создания изображений пальм.
